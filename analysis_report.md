# 台新新光集團合併新聞分析報告

分析時間：2026-04-18 17:17:19

### 事件總結

核心事件為台新金融控股股份有限公司（下稱台新金）與新光金融控股股份有限公司（下稱新光金）的合併案，導致新光金股東獲配台新新光金股份。此合併案已於2025年7月完成新股權利證書的上市買賣。合併後，主要衍生出對原新光金股東所獲配股份對價之股利所得的稅務處理需求，若其原始取得成本高於平均出資額，可申請更正。截至2026年第一季，合併後的「台新新光金」已展現強勁的營運成果，獲利創同期新高。

### 最新進度

最晚（最新）的合併進度顯示為2026年4月15日。依據該日發布的資訊，台新新光金控在2026年第一季的稅後淨利創下歷年同期新高，表現強勁。同時，其旗下的新光人壽在2026年3月單月稅後淨利成功轉虧為盈，首季累計獲利亦顯著。在股東稅務方面，原新光金普通股股東若欲舉證原取得成本超過平均出資額10.259元以更正股利所得，需於2026年3月31日前洽台新證券股務代理部辦理，或可於申報2025年度綜合所得稅時，自行於2026年6月10日前向戶籍地國稅局申請修正申報金額並繳稅。

### 風險與機會

**機會點：**
1.  **合併綜效顯現與獲利能力提升：** 台新新光金控在2026年第一季獲利創下歷年同期新高，顯示合併後的整合效益與營運綜效已逐漸顯現。尤其新光人壽成功從去年同期虧損轉為獲利，並受惠於CSM穩定攤銷及經常性收益挹注，為集團整體獲利貢獻良多。
2.  **市場環境有利：** 金控業普遍受惠於台灣資本市場熱絡、降息預期使外幣資金成本下降、以及銀行子公司存放款與財富管理動能強勁等有利因素，有助於利息與手續費收入增長，為台新新光金控提供良好的發展環境。

**潛在風險：**
1.  **稅務處理複雜性與股東關係管理：** 合併涉及前新光金股東股利所得的更正程序，流程較為複雜且設有期限。若未能有效溝通與協助，可能對股東關係及行政效率造成影響。
2.  **法規遵循與營運整合挑戰：** 新光人壽在2026年第一季因不動產與個資管理缺失而遭受金管會裁罰，反映合併後在營運標準化及法規遵循方面仍有待加強，可能存在潛在的整合風險。
3.  **地緣政治與市場波動風險：** 中東衝突等外部地緣政治事件已對壽險業的金融資產評價造成衝擊，儘管新光人壽在此波動中維持獲利，但整體集團仍將持續面臨國際情勢變化對投資收益的潛在影響。

**投資/觀察建議：**
整體而言，台新新光金控展現出強勁的合併後獲利動能與壽險業務的顯著改善，值得投資者正面關注。然而，應密切追蹤其在處理股東稅務議題上的效率與透明度，以及在內部營運整合與法規遵循上的持續進展。同時，鑑於金融市場易受外部事件影響，集團對風險的控管能力，特別是其投資組合在全球宏觀不確定性下的表現，將是評估其長期價值的關鍵指標。
